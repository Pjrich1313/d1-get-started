# Changelog

All notable changes to this project will be documented in this file.

## [0.0.0] - 2026-05-20

### Added

- Initial Cloudflare Workers starter for the `pamela` project.
- D1-backed example API endpoints and database schema for customer data.
- API key protection for `/api/` routes with deployment guidance.
- Blockchain webhook example for storing on-chain event payloads.
- Binary search tree implementation and example documentation.
- Supporting project guides for API security, blockchain usage, and
  performance considerations.
