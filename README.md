# pamela

A Cloudflare Workers project using D1 database.

## Getting Started

This is pamela, a starter template for working with Cloudflare D1.

### Prerequisites

- Node.js installed
- A Cloudflare account
- Wrangler CLI installed

### Installation

```bash
npm install
```

### NPM Configuration

You can customize npm behavior using configuration commands:

```bash
# Set configuration values using key=value syntax
npm config set key=value [key=value...]

# Alternatively, use the shorthand 'set' command
npm set key=value [key=value...]
```

**Common configuration examples:**

```bash
# Set the npm registry
npm config set registry=https://registry.npmjs.org/

# Configure a proxy
npm config set proxy=http://proxy.company.com:8080

# Set exact version saving
npm config set save-exact=true

# View current configuration
npm config list
```

The project includes a `.npmrc` file with recommended default settings.

### Development

```bash
npm run dev
```

### Deployment

```bash
npm run deploy
```

## About pamela

pamela demonstrates the basics of working with Cloudflare D1 database in a Workers environment.
